# Product Definition: Olympus Fleet

The **Olympus Fleet** is a distributed, sovereign AI development and execution environment. It is designed to run "GCP on a Workstation" while maintaining strict adherence to the **FLEET_GENESIS** architectural grammar.

## Core Objectives
1. **Sovereignty**: Complete local execution of cloud-native patterns (Vault, Pub/Sub, Cloud Run).
2. **Correctness**: "Correct-by-Construction" patterns using Go 1.25 and Dagger pipelines.
3. **Synergy**: Seamless handoffs between human developers and AI agents (Antigravity + Gemini-CLI).

## Key Components
- **Olympus2**: The core management layer and substrate.
- **OlympusForge**: The sovereign factory (CI/CD + Tooling).
- **OlympusGrammar**: The IR and nomenclature standard.
- **OlympusAssurance**: The testing and validation framework.
